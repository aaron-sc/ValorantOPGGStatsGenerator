# User Guide

## 1. Build a Team Dashboard
1. Open the app and upload a team list with Riot IDs.
2. Select the seasons to scrape.
3. Click **Start Scrape** and wait for completion.
4. You'll be redirected to the team dashboard.

## 2. Filters
### Map Filters
Click **Map Filters** to enable/disable maps.
By default, only the current competitive pool is enabled.

### Pick/Ban Planner
Pick order uses adjusted win% (raw win% smoothed toward 50% for small samples).
Ban order is the lowest adjusted win%. Only maps with at least 3 games are prioritized.
Confidence labels are based on games: High (>=12), Medium (>=6), Low (<6).

## 3. Compare Teams
Use **Compare Teams** on the landing page to generate a comparison dashboard.

## 4. Export
Snapshots and comparisons can be exported as JSON/ZIP for sharing.
Use **Export Snapshot (.png)** on team and comparison dashboards to share a full-page image.
